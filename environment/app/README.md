# scientific-computing GF(256) channel-coding numerics

Scientific-computing numerical analysis driver for Reed-Solomon channel coding
and discrete mask-penalty minimization over GF(256) two-dimensional binary
module lattices (ISO/IEC 18004 Model 2 algebraic specification): mixed-mode
segmentation dynamic programming, finite-field generator polynomials, block
interleave geometry, and combinatorial mask selection with exact syndrome
checks and external matrix decode verification.

Contract docs: /app/docs/*.md. Numerical defaults: /app/config/composer.toml.
Rebuild: /app/docs/rebuild-contract.md. Instrument: /app/bin/qr-composer.
