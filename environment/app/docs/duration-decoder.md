# Duration Decoder

The model uses an elapsed-duration state in addition to the clinical state. The duration caps are quiet 4, flow_limited 3, and apnea 4. An expanded state is written as (state, age), where age starts at 1 when a clinical state is entered and is clamped at that state's cap. Expanded states are ordered by clinical state order and then increasing age.

Start probabilities use additive smoothing of 0.5 over the first clinical state of every training sequence. Each clinical state emits a two-component mixture of full four-feature Student-t distributions with fixed degrees of freedom 5. The mixture components are ordered 0 then 1. Their fitted weights are positive and sum to one within each clinical state. The labeled initialization, calibration responsibilities, and re-estimation rules are fixed in /app/docs/adaptation-policy.md.

For one component with mean vector mu, let S(f,g) be the population second moment for features f and g minus mu(f) * mu(g). The covariance diagonal is S(f,f) + 0.0001. Every off-diagonal covariance is 0.80 * S(f,g). This calculation is performed independently for every state and component. Each resulting covariance matrix is symmetric and positive definite.

For every labeled training row that has a following row, determine its clamped age within the current same-state run. If the following state matches, add one to that state's continue count at this age. Otherwise add one to its exit count at this age and one to the exit-destination count for the following state. Do not add duration counts for a final row with no following row. For each state and age, duration_continue_probability is (continue_count + 0.5) / (continue_count + exit_count + 1.0). For each from-state, exit_destination_probability for an other-state is (destination_count + 0.25) / (sum of destination counts to other states + 0.25 * 2).

At an expanded state (state, age), a same-state next row goes to (state, min(age + 1, cap)) with duration_continue_probability. A different next clinical state goes to (other_state, 1) with (1 - duration_continue_probability) multiplied by exit_destination_probability for that other state. Compute every component's multivariate Student-t density through a lower-triangular Cholesky factor L where covariance = L * L-transpose: solve L * y = x - mu, use the quadratic q = y-transpose * y, and use 2 * sum(log(L(i,i))) as the log determinant. With d = 4 and nu = 5, the natural-log component density is lgamma((nu+d)/2) - lgamma(nu/2) - 0.5 * (d * ln(nu * pi) + log_determinant) - ((nu+d)/2) * ln(1 + q/nu). For a state emission, add ln(component_weight) to each component density and take logsumexp across components 0 and 1. Initial expanded states are only (state, 1), scored with the matching start probability and first-row mixture emission.

Run Viterbi in natural-log space over expanded states. For every candidate comparison, replace the current best predecessor only when candidate is greater than best plus 1e-12. A candidate within 1e-12 is a tie, so keep the earlier predecessor in expanded-state order. Apply the same comparison when choosing the final expanded state. Collapse the selected expanded path to clinical states for predictions and validation metrics.

Run forward-backward in natural-log space over the same expanded transition system. The first forward score uses the start probability and first emission. The final backward score is zero for every expanded state. Sequence log likelihood is logsumexp of the final forward scores. At row t, the raw posterior mass of an expanded state q is exp(forward_t(q) + backward_t(q) - sequence_log_likelihood); an impossible or non-finite mass is zero. Sum raw masses in expanded-state order into M_t(state), one value for each clinical state. Let Z_t be M_t(quiet) + M_t(flow_limited) + M_t(apnea), added in that state order. The clinical posterior is M_t(state) / Z_t. Always perform this normalization, including when the raw masses are already close to one. Posterior probabilities must sum to one, and entropy is -sum(p * ln(p)) with a zero-probability term contributing zero.
