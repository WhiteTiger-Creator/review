# Sensor Adaptation Policy

The labeled training studies establish fixed baseline sufficient statistics. The unlabeled calibration studies from --adapt then adapt the model before any validation or inference sequence is decoded. Use exactly three anchored expectation-maximization rounds. The labeled baseline is retained in every round; do not relabel calibration samples and do not use validation or inference traces during adaptation.

Initialize two emission components within each clinical state from the labeled rows. For one state, sort that state's labeled spo2_drop values in ascending order. Its median is the middle value for an odd count and the average of the two middle values for an even count. Assign a labeled row to component 0 when its spo2_drop is less than or equal to that state median, and otherwise assign it to component 1. Keep separate labeled component weights, feature sums, and ordered feature-pair cross-sums. These component assignments remain fixed throughout adaptation.

At the expectation step, run the forward-backward calculation from /app/docs/duration-decoder.md on every calibration sequence with the current model. For calibration row t, calculate gamma_t(state) as the normalized clinical posterior M_t(state) / Z_t defined in that document. Do not use the unnormalized clinical mass as gamma. On the first row of a calibration sequence, add the raw posterior mass of expanded state (state, 1), divided by that same row's Z_0, to the calibration start count. For a component, calculate its conditional responsibility within that state as exp(ln(component_weight) + ln(component_density) - ln(state_mixture_emission)). Add gamma_t(state) multiplied by that responsibility to the matching component's calibration emission weight, and multiply the same value by each feature and every ordered feature pair for its component feature sum and cross-sum.

For each adjacent calibration pair t and t + 1, use the expanded-state transition posterior xi_t(from, to). If from and to have the same clinical state, add xi_t to the continuation count at from's elapsed age. If their clinical states differ, add xi_t to the exit count at from's elapsed age and to the exit-destination count from the first state to the second. xi_t(from, to) is exp(forward_t(from) + log_transition(from, to) + log_emission(to, row t + 1) + backward_t+1(to) - sequence_log_likelihood). Impossible transitions contribute zero.

At every maximization step, combine the fixed labeled count or moment with the current round's expected calibration value. Start probabilities use (labeled_start_count + calibration_start_count + 0.5) / (training_sequences + adaptation_sequences + 1.5). For each state and age, continuation probability uses (labeled_continue_count + calibration_continue_count + 0.5) divided by labeled and calibration continuation plus exit counts plus 1.0. For each exit destination, use its combined count plus 0.25 divided by the sum of combined counts to the other two states plus 0.5.

For an emission component, its combined weight is its labeled component weight plus its expected calibration component weight. The state weight is the sum of its two combined component weights. Its mixture weight is (component_combined_weight + 0.25) / (state_weight + 0.50). Its mean is its combined feature sum divided by its combined component weight. For each component feature pair, form the combined population second moment from its cross-sum and subtract the two component means' product. Add 0.0001 on the covariance diagonal and multiply every off-diagonal by 0.80. The next expectation step uses this re-estimated full-covariance mixture model. Record the total natural-log likelihood of all calibration sequences before each re-estimation in adaptation_log_likelihood.
