# Example case files

These sample case files exercise the toolkit front end and readers. They carry case
inputs only; no reference censuses are provided with them. Use them to confirm the
toolkit builds, fetches the conductance operator and runs end to end.

- `slab_sweep.cse` three bare cases at rising shifts
- `hotspot_patch.cse` two cases carrying conductance side terms only
- `sink_retrofit.cse` two cases carrying terms on both sides of the pair
