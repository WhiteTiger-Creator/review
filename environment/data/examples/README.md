# Example query files

These sample query files exercise the toolkit front end and readers. They contain query
inputs only; no reference counts are provided with them. Use them to confirm the toolkit
builds, fetches the operator and runs end to end.

- `case_a.qry` a single unmodified case and one two term modification
- `case_b.qry` three cases with one term each at different shifts
- `case_c.qry` a case with several terms of mixed sign weights
