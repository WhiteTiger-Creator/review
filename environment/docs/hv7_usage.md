# hv7

Refresh the thermal kernels, then emit the sealed certificate.

```
/app/environment/tools/build.sh
/app/environment/tools/hv7 --corpus /app/environment/data/pack_c --out /app/output/invariant_bundle.yaml
```

Stage files under `/app/output/stage` are early gauges only. They do not prove closed algebra. Schema, digests, residuals, journal stitch, and seal formulas are normative in `/app/environment/docs/t_policy.rst`.
