The toolkit works on a large sparse conductance operator from a thermal finite element model, not yet on disk. Run /app/scripts/fetch_operator.sh to download it to /app/operator.mtx. Each case names a rate shift and patch terms on both sides of the thermal pair, and asks how many relaxation modes of that patched model have a rate strictly below the shift, the part of its generalized spectrum lying under that shift. The census still comes back from /app/src/counter.c as a placeholder. Compute it numerically and exactly for every case, with /app/mode_census run on the operator and a case file printing one census per line on standard output in case order. The model carries over a million degrees of freedom, so forming or diagonalizing a patched member is not feasible, and the run has to finish inside the wall clock the grading run allows. Censuses are checked on unseen cases for exact equality.
