A large sparse symmetric operator from a thermal model backs the toolkit in this directory, but it is not on disk yet. Run /app/scripts/fetch_operator.sh to download it to /app/operator.mtx. Each query case names a shift and a symmetric low rank modification of the operator, and asks how many eigenvalues of the modified operator are strictly below the shift. The routine in /app/src/counter.c that is meant to compute this count is unfinished and returns a placeholder. Make it compute, for every case, the exact number of eigenvalues below the shift, so that /app/spectral_count run on the operator and a query file reports every count correctly. The operator has over a million degrees of freedom, so forming or diagonalizing the modified operator directly is not feasible. The counts are checked on unseen query cases for exact equality.
